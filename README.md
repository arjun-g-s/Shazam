# Shazam Audio Fingerprinting Project

A project to generate **audio landmarks** from songs and store them in a **SQLite database** for fast song matching.

---

## **Prerequisites**

- [Download DB Browser for SQLite](https://sqlitebrowser.org/) to view and manage the database.  
- [Git](https://git-scm.com/) installed to clone the repository.  
- Python 3.x (if running scripts to generate landmarks).  
- Required Python libraries (install via pip):

```bash
pip install numpy scipy matplotlib soundfile sqlite3
